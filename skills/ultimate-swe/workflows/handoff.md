# 7. Session Handoff

## Objective

Preserve project state for seamless continuation.

---

## Required Output: PROJECT_STATE.md

Update or create:

- Completed features
- Current work
- Next steps
- Known risks
- Open questions

---

## Key Activities

- Document completed work
- Record unfinished tasks
- List blockers
- Summarize progress clearly
- Provide restart instructions

---

## Rule

A session is incomplete without a proper handoff document.
